Single Round Match 663 - Round 1

easy:

设走的总长度为sum，最长的那一步为max。
满足两个条件才能达到要求：
1、sum >= X
2、sum - max + X >= max
第一个条件是很显然的。
而关键是超出的部分能不能折回来。
第二个条件:
根据两边之和大于第三遍这个定理YY一下，多边形如果最长边小于等于其他边的和，多边形肯定是存在的的。

hard

2-sat，和SRM630的hard一样的做法。

做法:

对于每个素数单独分析。

建图：

每个数先建k个点，表示这个数的素因子中，当前素数出现大于等于k次。
每个点的相对点就表示当前素数出现次数小于k次。
然后连一下边：比如选了大于等于k次，肯定要选大于等于k-1次。

对于gcd和lcm：

设两个数的gcd中当前素数个数为x；a中包含当前素数个数为na，b中包含当前素数个数为nb
1、a >= x， b >= x。
2、如果 a >= x+1，那么 b < x+1；如果 b >= x+1，那么 b < x+1。

lcm也是一样的。
